# プロフィール画像メーカー (profile-maker)

## 🌟 概要

`profile-maker` は、学生部 新歓用のプロフィール画像をブラウザだけで作成できるシンプルなWebアプリです。画像のアップロード、トリミング、各種プロフィール項目入力、リアルタイムプレビュー、PNGダウンロードまでをひとつの画面で行えます。

- 開発者: 永田大悟
- 対象: 首都圏新歓／部活・サークル紹介用プロフィールカード
- 実行環境: モダンなブラウザ（Chrome、Edge、Firefox、Safariなど）

## 🛠️ 機能

1. プロフィール写真をファイルから選択
2. `Cropper.js` によるトリミング（縦横比固定）
3. 名前・学校・出身・趣味などのテキスト項目入力
4. デザインテンプレートにテキストと画像を合成
5. その場でプレビュー表示
6. `画像をダウンロード` で `profile_card.png` を生成
7. ネイティブ共有APIが利用可能なら共有機能も利用可

## 📁 ファイル構成

- `index.html` - 本体のUIとロジック
- `template.jpg` - 背景テンプレート（デフォルト項目）
- `README.md` - プロジェクト説明

## 🚀 使い方

1. このリポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く
3. 「プロフィール写真」を選択し、表示されたモーダルで切り抜き
4. 各テキスト項目を入力
5. プレビューを確認
6. 「画像をダウンロード」ボタンをクリック

## 📌 テンプレート調整

- `index.html` 内の `textConfig`、`photoConfig` を変更すると、出力位置・フォントサイズを調整できます。
- `bgImage.src = 'template.jpg'` の背景画像ファイル名を変更可能

## 💡 カスタマイズ例

- 新しい項目追加（例: 趣味、特技、好きな一冊など）
- フォント・カラーの変更
- 出力サイズを高解像度に変更
- WebUIのレスポンシブ対応（スマホ・タブレット）
